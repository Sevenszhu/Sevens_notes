# 从零开始学数字电子技术

# 第一章 数字电路入门

### 第一节 从模拟电路到数字电路

- **模拟信号**： 表示在时间和数值上变化是连续的量（**模拟量**）的信号是模拟信号（信号大小随时间做连续变化）*我感觉这里的随时间只是描述其**连续性**，不表明信号大小和时间有相关性*
- **模拟电信号**： 将模拟量转换成电流电压等电学量。
- **模拟电路**： 用于传递、加工和处理模拟电信号的电路。
    - 模拟信号的幅度大小代表了其本身性质，所以放大缩小需要线性地。若一模拟信号幅度大，对放大器（增强并复现）要求高。
    - 模拟电路的干扰信号常常是模拟信号，一旦干扰，难以分开，故要有强抗干扰能力。 
    -  每处理一套信号，需要一套电路与之对应，使得模拟电路庞大复杂，模拟电路不便对信号进行深层的复杂处理
- **数字信号**：表示在时间和数值上均是离散的（变化在时间上是离散的）量的信号称为数字信号。数字波形表示数字信号，数字波形用逻辑电平表示（对电压高低的标准化定义：高电压表示1，低电压表示0）（分为周期性和非周期性
- **脉冲波形**：只有两个离散值的数字波形（用电压值表示）
- **非理想数字波形**： 数字波形现实中不能立即上升或下降，要经历一段时间
- 定量描述非理想数字波形的指标：
    - **脉冲周期T**： 周期性重复的脉冲序列中，两个相邻脉冲的时间间隔
    - **频率f**： 单位时间内脉冲重复的次数
    - **脉冲幅度$U_m$**： 脉冲电压最大变化幅度
    - **脉冲宽度$t_w$**：脉冲超过临界值开始，到下降到临界值为止的一段时间
    - **上升下降时间**：脉冲分别从最低点上升到最高点的时间和从最高点下降到最低点的时间
    - **脉冲占空比q**： $q = \dfrac{t_w}{T}  \times 100\%$ （脉冲时间占整个周期的百分比）
- **二值数字逻辑**： 用0和1来表示自然界中彼此相关却对立的两种状态 eg. 是与非，真与假
- **逻辑电平** ： 二值数字逻辑在数字电路的应用，高电压1，低电压0
- **时序图**： 表示相互时间关系的多重数字波形图，其中的每一个波形被称为是时序信号
    - **时钟脉冲CP**：数字系统中的时间参考信号，一般用石英晶体振荡器产生，波形是对称波形。
- **数字电路**：传递、加工和处理数字信号的电路，是一种脉冲电路（如果数字信号在有和无之间快速转换，那么电路将输出一系列矩形脉冲）
- 数字电路是对数字信号进行逻辑运算和数字处理，数字电路具有以下一些特点：
    - 采用二值信息——高电平和低电平，只有两种取值
    - 数字电路的晶体管只工作在饱和和截止两种状态，即开关两种状态。
    - 数字电路由几种最基本的单元电路组成的，这种单元电路对元件的精度要求不高，允许有较大的误差范围
    - 对于数字电路，人们主要研究输入输出信号的状态之间的逻辑关系，
    - 数字电路的研究内容
        - **逻辑分析**： 分析已有电路的逻辑功能
        - **逻辑设计**： 根据逻辑功能的要求设计出满足该逻辑功能的电路
- 数字电路的分类

    - 电路类型分：
        - **组合逻辑电路**：输出只与当时的输出有关
        - **时序逻辑电路**： 输出不止与当时的输出有关，还与电路的原来状态有关

    - 集成类型分：
        - 分为小规模、中规模、大规模和超大规模

    - 电路所用器件分：
        - 双极型：晶体三极管型
        - 单极型 :  场效应管型
- **数字系统**： 
    - **输入系统**： 将模拟量转换为数字量
    - **数据处理器和数据控制器**： 控制系统内部各部件的工作，使他们按照一定程序操作
    - **输出系统**： 将数字量转为模拟量。
- **模拟信号的数字化表达**： 将时间上连续变化的模拟量变成脉冲有无的数字量（就是将模拟量变成数字量）
- **模拟系统的缺点**： 
    - 模拟系统在记录、编辑、传送和回放的过程中，不可避免地混进杂音产生各种干扰，这些有害成分往往难以与有用信号分开，损伤原信号
    - 信号储存过程中，记录媒体的信噪比决定了信号被记录下的信噪比，因此模拟系统中对记录媒体材料的要求很高
    - 记录媒体的转换器如果存在非线性失真，这种失真无法通过电路解决
    - 对重放过程中的机械要求很高，如果工作状态不稳定，会有抖动失真
- **数字系统的优点**：
    - 数字系统记录的形式是二进制编码，只要判断是0还是1，对记录媒体要求低
    - 数字系统只有记录0和1，只要保证0和1之间的有足够的电平差，数字电路就能方便地把他们识别和分离，说明数字电路的抗干扰性强
    - 采样频率和量化位数确定，电路的性能极限就确定了，且不容易被改变，数字系统性能的重现性是很高的，模拟系统的因为不稳定因素，重现性能是不稳定的
    - 编码后的信号和处理该信号过程中混入的干扰信号性质不同，两者可以轻松地被分离，因而可以轻松地去除干扰信号
    - 数字记录系统中，重放系统设有时基校正电路，系统不稳定现象不会引起抖动失真的问题
    - 数字系统中的数字系统只是在低电平和高电平两种状态下交替变化，不必要求记录系统和重放系统中的换能器具有良好的线性
    - 实现相同功能时，数字信号没有模拟电路那么复杂

### 第二节 数制的转换及运算

- **数制**： 就是计数法、进位制
- **基**： 数制所使用的数码的个数，譬如十进制就是十个：0123456789
- **权**： 数制的每一位所具有的值
- **位权**： 某一位一个单位所代表的实际数值大小，$位权 = 基数^{位置序号}$
- 构成数字电路的基本思路就是把电路的状态和数码对应起来，十进制的是个数码，往往需要对应十个严格区分的电路状态，这在技术上会很困难，且不经济
- 二进制的优点：
    - 只需要两种状态表示数字，容易实现，稳定可靠。eg. 磁化和未磁化，导电和未导电
    - 二进制的运算规则简单，运算操作方便
    - 二进制容易实现逻辑运算，0表示false，1表示true

- 二进制的缺点：位数多，不容易书写和记录。所以资料中常常使用8进制或者16进制来表示二进制

### 第三节 编码的传输和读取

- 数字系统中的信息分为两类：数值信息和文字符号信息（包括控制符）。文字符号信息往往采用一定位数的二进制码表示，这个二进制码称为**代码**
- **数位**： 码的位叫做数位。 eg. 100101这个代码有6个数位（bit）。 其中，二进制数位也叫bit（比特）

- **字**： 表示某一个数值或字符的二进制数。在数字电路中，所有的信息都是以二进制代码表示的
- **字长**： 一个字的二进制位数称为字长。微控制器的字长是有1位、4位、8位和16位等
- **字节**： 一组二进制位形成的计算机的一种存储单位，它可以表示一个字符，通常一个字节为8个二进制位
- **二进制-十进制编码（BCD码）**： 用一个四比特二进制码来表示十进制的编码（用四位数表示十进制中的0-9）
    - 8421BCD码：
        - 8421BCD码的权自左至右是8、4、2、1
        - 其他BCD码也是这样
        - 还有5421、2421码等

- **ASCII码**： 用有特定规则的二进制编码来表示各种字母和符号

- **原码**： 机器不认识$+-$  符号，若表示正数，在二进制的最高位后加0，表示负数在二进制最高位加1
- **反码**： 正数的反码和其原码相同，负数的反码等于其除最高位各位求反。
- **补码**： 正数的补码与其原码相同，负数的补码是将其原码求反码，然后在最低位加1
- **奇偶校验码**： 检测存取、运算和传输过程中，错误将0和1写错的情况，分为信息位和奇偶校验位。

- **计算机数据的传播途径**： 网络电缆，固体磁盘，无线电波。信息用代码表示后传输，传输过程中的信息要有统一的协议。

- **数据的传输方式**： 

    - **串行方式**： 一根导线，一个共同接地端，每传输一个数据需要一个时钟周期

        - eg. 传输一个字节（8位）需要分八次
        - **共同接地端**： 连接到同一个电位参考点，作用：
            - 统一电位基准
            - 稳定信号传输

        - **时钟周期**： 系统时钟信号的基本单位，用于同步收发双方的操作节奏

    - **并行方式**： 二进制各位同时传输，这样要求导线的数目跟二进制数码的数位相同。

- **波特率**： 码的传输速度，每秒所传输数据的位数，它的单位是位/秒（bit/s）。波特率是衡量传输通道频宽的指标，和传输数据的速率并不一致。因为数据传输中有起始位和终止位，他们也占字符。
